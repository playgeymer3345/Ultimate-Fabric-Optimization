# ⚠️ ВНИМАНИЕ ДЛЯ ВСЕХ ИГРОКОВ / NOTICE FOR ALL USERS

### 🔒 Если кнопка "New issue" горит серым и не нажимается:
Вам **ОБЯЗАТЕЛЬНО нужно войти в свой аккаунт GitHub или создать новый**! 

Платформа GitHub запрещает гостям без аккаунта отправлять баг-репорты, чтобы защитить репозитории от спам-ботов. Как только вы зайдете в аккаунт, кнопка сразу станет активной и зеленой.

* [Нажмите сюда, чтобы войти в аккаунт (Sign in)](https://github.com)
* [Нажмите сюда, чтобы зарегистрироваться (Sign up)](https://github.com)

---

### 🐛 Как правильно отправить баг-репорт (после входа):
1. Нажмите активировавшуюся кнопку **New issue**.
2. Выберите шаблон **Сообщение об ошибке / Bug report**.
3. Опишите проблему и обязательно прикрепите файлы `latest.log` или `crash-report.txt`.
