# ⚡ Ultimate-fabric-optimization

**Ultimate-fabric-optimization** — это высокотехнологичная сборка модов для Minecraft, созданная для максимального FPS, устранения фризов и обеспечения идеальной плавности геймплея.

## 🐛 Сообщить о баге (Инструкция)

1. Перейдите на вкладку **Issues** вверху страницы.
2. Войдите в свой аккаунт **GitHub** (или зарегистрируйтесь).
3. Нажмите зелёную кнопку **New issue** и выберите подходящий шаблон.

## 📜 О проекте

Проект использует и комбинирует лучшие современные моды на оптимизацию и распространяется под открытой лицензией Mozilla Public License 2.0 (MPL-2.0).

⚠️ **ВАЖНОЕ ПРИМЕЧАНИЕ:** Содержимое сборки постоянно дорабатывается и обновляется. Некоторые моды могут удаляться из-за несовместимости, а новые — добавляться для улучшения производительности. В связи с этим реальный состав файлов в игре может временно отличаться от текстовых описаний как на странице Modrinth, так и здесь, на GitHub. Актуальный состав сборки всегда находится в самом скачиваемом пакете модов.
