### 📊 Состав модов

#### ⚡ Увеличение FPS и Рендеринг
* **Sodium** — Радикально поднимает FPS и снижает нагрузку на видеокарту.
* **Sodium Extra** — Дополнительные настройки графики в меню Sodium (анимации, частицы).
* **Reese's Sodium Options** — Делает меню настроек Sodium очень красивым и удобным.
* **ImmediatelyFast** — Оптимизирует рендеринг текста, шрифтов, табличек и полосок здоровья.
* **BadOptimizations** — Убирает фоновые лишние вычисления рендеринга неба и интерфейса.

#### 🎯 Отсечение невидимого
* **Entity Culling** — Пропускает рендеринг мобов и сундуков, которые находятся за стенами.
* **More Culling** — Оптимизирует просчет листьев деревьев, травы и жидкостей.

#### 🧠 Логика, Движок и Память
* **Lithium** — Полностью переписывает физику игры и ИИ мобов для плавности мира.
* **ModernFix** — Исправляет баги движка Minecraft и убирает утечки памяти.
* **FerriteCore** — Радикально сжимает оперативную память, уменьшая потребление RAM.
* **Better Block Entities** — Оптимизирует рендеринг сложных блоков (сундуков, спавнеров).
* **Too Many Entities** — Стабилизирует игру при огромном количестве предметов на экране.

#### 🗺 Генерация мира и Сеть
* **C2ME** — Многопоточный движок для идеально плавного полета и прогрузки чанков.
* **Chunky** — Позволяет заранее сгенерировать карту мира для экономии ресурсов.
* **Dynamic FPS** — Снижает FPS до 15 кадров, когда игра свернута.
* **Fast Server Pings / Fast IP Ping** — Оптимизирует сетевой код для снижения пинга.

#### 🔊 Аудио-атмосфера и Интерфейс
* **Sound Physics Remastered** — Добавляет реалистичное физическое эхо и реверберацию звуков.
* **AmbientSounds** — Наполняет мир Minecraft живыми звуками природы.
* **Presence Footsteps** — Полностью перерабатывает и улучшает звуки шагов по блокам.
* **Iris Shaders** — Самый быстрый и современный движок для поддержки шейдеров.
* **Mod Menu** — Добавляет в главное меню удобный список всех установленных модов.

### 📊 Mod List

#### ⚡ Performance & Graphics
* **Sodium** — Radically boosts FPS and reduces GPU load.
* **Sodium Extra** — Adds extra graphical settings to the Sodium menu (animations, particles).
* **Reese's Sodium Options** — Makes the Sodium options menu clean and searchable.
* **ImmediatelyFast** — Optimizes rendering of text, signs, and entity HUDs.
* **BadOptimizations** — Removes unneeded background calculations for sky and UI rendering.

#### 🎯 Culling
* **Entity Culling** — Skips rendering entities and chests hidden behind walls.
* **More Culling** — Optimizes culling for leaves, grass, and fluids.

#### 🧠 Engine & RAM
* **Lithium** — Overhauls game physics and mob AI for better server/client ticks.
* **ModernFix** — Fixes bugs in the Minecraft engine and eliminates memory leaks.
* **FerriteCore** — Dramatically reduces memory usage by compacting RAM data.
* **Better Block Entities** — Optimizes rendering for tile entities like chests and spawners.
* **Too Many Entities** — Stabilizes the game when too many items or entities are present.

#### 🗺 World & Network
* **C2ME** — Multi-threaded chunk engine for stutter-free world loading.
* **Chunky** — Pre-generates world chunks to prevent lag spikes during exploration.
* **Dynamic FPS** — Lowers FPS to 15 when the game window is unfocused or minimized.
* **Fast Server Pings / Fast IP Ping** — Optimizes network code for faster server pings and lower latency.

#### 🔊 Audio & UI
* **Sound Physics Remastered** — Adds realistic sound physics, echoes, and cave reverberation.
* **AmbientSounds** — Fills the world with realistic ambient sounds (birds, wind, caves).
* **Presence Footsteps** — Completely overhauls footstep acoustics for every block type.
* **Iris Shaders** — Modern and fast shader engine with OptiFine shader pack support.
* **Mod Menu** — Adds a clean mod listing screen to the main menu.
